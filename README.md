## 📖 Project Overview

This project is based on the study and analysis of open-source software.
Git is selected to understand its origin, license, Linux usage, and ecosystem.

The project also includes shell scripting tasks performed in Ubuntu (WSL).

---

## ⚙️ Scripts Included

### 🔹 Script 1 – System Identity

Displays system information like kernel version, user, uptime, and date.

### 🔹 Script 2 – Package Inspector

Checks whether Git is installed and shows its version.

### 🔹 Script 3 – Directory Audit

Shows directory size and permissions using loop.

### 🔹 Script 4 – Log Analyzer

Counts the number of "error" lines in log files.

### 🔹 Script 5 – Manifesto Generator

Generates a personalized open-source statement using user input.

---

## 💻 How to Run Scripts

chmod +x script1.sh
./script1.sh

(Same for all scripts)
