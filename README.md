## 📖 Project Overview
This project focuses on the study and analysis of open-source software. Git has been selected to understand its origin, licensing, usage in Linux, and its role in the open-source ecosystem.

The project also includes practical implementation through shell scripting using Ubuntu (WSL).

## ⚙️ Scripts Included

🔹 Script 1 – System Identity  
Displays system information such as kernel version, user, uptime, and date.

🔹 Script 2 – Package Inspector  
Checks whether Git is installed and displays its version.

🔹 Script 3 – Directory Audit  
Shows directory size and permissions using loops.

🔹 Script 4 – Log Analyzer  
Counts the number of "error" lines in log files.

🔹 Script 5 – Manifesto Generator  
Generates a personalized open-source statement based on user input.

## 💻 How to Run Scripts
Use the following commands in the terminal:

chmod +x script1.sh  
./script1.sh  

(Repeat the same steps for other scripts.)

## 🐧 Linux Environment
Platform: Ubuntu (WSL)  
Commands used: uname, whoami, uptime, dpkg, grep, du, ls, awk
